Smart Bag Controller

A smart anti-theft and monitoring system for everyday backpacks

Overview

The Smart Bag Controller is an intelligent system designed to enhance the safety and usability of backpacks. It uses sensors and smart logic to detect unauthorized access and notify the user instantly.

Development Environment

Built and tested using Android Studio with real-time UI rendering.

Key Features

  • Zip detection using distance sensor
  • Instant alert system for unauthorized access
  • Real-time monitoring
  • Compact and efficient design

How It Works

The system continuously monitors the position of the bag’s zip using a sensor. If the zip is opened beyond a defined threshold, the system triggers an alert. This ensures immediate awareness of potential theft or tampering.

Technologies Used

  • Arduino (Microcontroller)
  • Ultrasonic Sensor
  • Embedded C / Arduino IDE
